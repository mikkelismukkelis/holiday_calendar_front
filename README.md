# Holiday / absences calendar

Interactive timeline for absences. Absences can be added, modified and removed. Timeline can be zoomed and moved.

## Frontend

- React
- MUI
- React-calendar-timeline
- Axios
- etc

## Backend

- MS SQL server
- Rest made with express, no authentication
